About the Model

NeuralTrade uses a Proximal Policy Optimization (PPO) reinforcement learning agent from Stable-Baselines3. The agent learns to trade a single stock by interacting with a custom OpenAI Gymnasium environment.

State Space (12 features)

Close Price Volume (normalized) EMA-20 EMA-50 RSI-14 MACD MACD Signal BB Width Daily Return Cash Ratio Shares Held Unrealized PnL

Action Space

BUY — Deploy all cash into shares
HOLD — Maintain current position
SELL — Liquidate all shares

Reward Function

Reward = % change in portfolio value − 0.1 × drawdown penalty. This encourages the agent to grow wealth while penalising large losses.

Architecture

MlpPolicy with hidden layers [64, 64]. Trained with learning rate 3e-4, batch size 64, 10 epochs per update, γ=0.99, GAE λ=0.95.

⚠ Disclaimer: This tool is for educational purposes only. Past backtest performance does not guarantee future returns. Do not make real financial decisions based solely on this model.
